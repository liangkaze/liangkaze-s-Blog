# Ansible初始化配置

## 下载ansible

```shell
yum install ansible -y
dnf install ansible
```

## 设置ansible

先修改ansible的关键配置

```shell
[example@example01 ansible]$ vim /etc/ansible/ansible.cfg
[defaults]
# some basic default values...

inventory      = /etc/ansible/hosts
#library        = /usr/share/my_modules/
#module_utils   = /usr/share/my_module_utils/
#remote_tmp     = ~/.ansible/tmp
#local_tmp      = ~/.ansible/tmp
#plugin_filters_cfg = /etc/ansible/plugin_filters.yml
forks          = 5
#poll_interval  = 15
sudo_user      = example
ask_sudo_pass = false
ask_pass      = false
host_key_checking = False
#transport      = smart
#remote_port    = 22
#module_lang    = C
#module_set_locale = False

```

再在/etc/ansible/hosts中配置IP地址和主机名地址

```shell
[example@example01 ansible]$ vim /etc/ansible/hosts
# Ex 2: A collection of hosts belonging to the 'webservers' group

## [webservers]
## alpha.example.org
## beta.example.org
192.168.177.130
192.168.177.131

# If you have multiple hosts following a pattern you can specify
# them like this:

## www[001:006].example.com

# Ex 3: A collection of database servers in the 'dbservers' group

## [dbservers]
## 
## db01.intranet.mydomain.net
## db02.intranet.mydomain.net
## 10.25.1.56
## 10.25.1.57

# Here's another example of host ranges, this time there are no
# leading 0s:

## db-[99:101]-node.example.com
[myhosts]
example02 ansible_host=192.168.177.130
example03 ansible_host=192.168.177.131

```
